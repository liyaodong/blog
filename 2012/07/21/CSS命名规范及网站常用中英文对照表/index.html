一、样式文件命名规范

主要的 master.css
布局，版面 layout.css
专栏 columns.css
文字 font.css
打印样式 print.css
主题 themes.css

二、CSS注释

/CSS Document / //css文档
/Style for 项目名称/ //给某项目的样式
/Editor:hyy,10.24.2011/ //编辑人/日期
/Common style/ //公共样式
/User-defined global style/ //自定义全局样式
/Common Pages style/ //公共页面用样式
