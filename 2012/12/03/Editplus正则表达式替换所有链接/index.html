Editplus正则表达式替换所有链接

发表于 2012-12-03   |  

虽然文章很短，但是有时候可以给你省去很多时间。尤其是在。。窃取别人的劳动成果的时候。

Editplus打开，Ctrl+H

查找： <a[^>]*>
替换： <a href=”#”>

勾选正则表达式，如果需要，可以选择所有打开文件。。这是一个很邪恶的选项。

效果
<a href=”http://www.rs99.cn“ target=”_blank”>人生久久</a>
<a href=”http://www.rs100.cn“ target=”_blank” title=”rs100”>人生一百</a>
<a href=”http://www.sosohoo.com">sosohoo游戏攻略</a&gt;

替换后的结果
<a href=”#”>人生久久</a>
<a href=”#”>人生一百</a>
<a href=”#”>sosohoo游戏攻略</a>

恩，就是这个样子了。Editplus很强大，有时间了再写一个关于配置文件的，超爽的。
