创意包装设计欣赏

发表于 2012-08-18   |  

创意包装于商品，恰似画龙之后的点睛一笔，是功能性的必需品，更是灵性魅力俱佳的品质传播媒介。

两秒钟变个发型……做这种“理发师”，想必一定倍感自豪啦。

*Note earphones

音乐随“型”，让人从第一眼就能感受到乐符带来的悦动。

Designer: Corinne Pant.

*ScanWood Kitchen Equipment

原生的，健康的，环保的！那这组被新鲜泥土包围又留有根须的餐具兄妹们，恐怕称得上环保代言人了。

Design: Goodmorning Technology.

*Blush matches

如何向（几乎）所有男士、名正言顺地、宣传女士贴身内裤？这就是答案。

Made for Blush lingerie shop by BBDO Germany

  • Gubble Bum gum

无论你多健谈，知道吗，只要一粒Gubble Bum，就能封住他/她的嘴。

Designer: jjaaakk

*Gloji juice

有没试过在众人面前取下房顶的灯泡，拧“破”它，然后把“玻璃渣”倒进自己喉咙里……

  • Kleenex tissues

不得不承认，无论是色彩还是造型，它们都让人食欲倍增。

Designers: Jennifer Brock, Hiroko Sanders

*tPod tea

水能载舟亦能覆舟。这会儿，舟“沉”舟“浮”，可得我说了算。

*Energy drink

想让众人受惊？当着众人一饮而尽咯。

  • Stadium Nike box

Oh my lady Shoe box~终于了解什么叫别有洞天了。

*Natural fruit juices

纯天然果汁~当然得表里如一啦。

“Juice skin” by designer Naoto Fukasawa (Japan)

*Anti-Theft Lunch Bags

绝对的防盗，除食物主人之外，恐怕没第二个人敢拆开它。

*Royal tea

有时自娱自乐一下倒真是能让自己放松，例如，午后喝一杯别人的“洗澡水”。

Design: Don key Products (Germany)

*Anti-smoking package

不见棺材不掉泪。这次，是决定一边吸烟一边哭，还是决定直接掐断它们呢？

Designer: R.J. Reynolds

*”No More Sex” condoms

尽“性”法则：再一再二不再三。
