从零开始学习编程！Started to learn programming!

发表于 2012-08-14   |  

[caption id=”” align=”aligncenter” width=”600” caption=”program”][/caption]

如果你对编程有兴趣，如果你想试试开始学习编程，那么这篇文章将会对你有一定的作用~如果你之前认为编程是很无聊的，就像C语言老师上课讲的那样，那么，你错了。这里可以让你开心学习编程，Have fun ~ 一起来吧

在codecademy这个网站上从零开始

[caption id=”attachment_609” align=”aligncenter” width=”700” caption=”codecademy”][/caption]

>>Codecademy

点击进入后，按照控制台中的提示，开始学习。什么？控制台？不知道控制台是什么？没关系，就是网站首页左侧那个框。好了，按照上面说明的，一步一步开始吧！

关于Codecademy：作为一款非常酷的在线编程学习网站，它向用户提供了一种全新的学习方式。在基于网络互动编程教程的基础之上，手把手帮助用户了解编程的基本原理，体味Codecademy的趣味性学习。

当然，除过Codecademy，还有一些其他的从零开始学习编程的网站

1、Processing

Processing是一个开源的编程语言及环境，主要面向创建图形、动画及交互的人群。这是让你上手写出代码最简单的一种方法。虽然是一个学习工具，但是也能完成令人惊讶的专业性工作。

>> Processing (Reference, Tutorials, Wiki, Forum, Inspiration)

Bloc (Ruby)

Bloc是一家帮助你轻松学习Ruby编程的创业公司，允许任何用户在浏览器中即时创建和部署Ruby应用，并能即时查看运行结果。

>> Bloc

从HTML开始

如果你从一开始就想做网站，那你最好从学习HTML开始。这是让你适应编程很好的途径，在你感觉习惯过后再学习CSS。当这些都学得差不多时，你就想试着学习JavaScript了。

>> W3Schools， 30 HTML Best Practices for Beginners，Learn HTML/CSS in 30 days

Kids Ruby

此前我们也曾报道过这个帮助孩子从小就能学习Ruby编程的网站。但即使你不是孩子，想从头开始学习编程，它也是可以给你带来帮助的。

>> Kids Ruby

Mozilla的开发者网络

在该网站上有一套循序渐近的教程和Demo演示集合，对编程学习很有帮助。同时，你可能对Google Code University也会感兴趣。

>> Mozilla: Developer Network 和 Google: Code University

P2PU

P2PU是一个完全自由的教、学社区，你可以这里报名学习任何你想学的内容。

>> P2PU

除此之外，网络上还有许多其它的应用以及编程相关的书籍、视频教程等内容，自己也可以找来这些内容不断的阅读并练习。如果能保持一颗求知的心，你终会找到适合自己的编程学习方法。

当然，在学习的过程中还会遇到一些问题，这个时候就要借助搜索引擎的力量了，Google、百度都可以搜索到一些蛛丝马迹，当然，也有一些专门的编程问答社区，比如德问。

>>德问

有意思的是，德问社区注册账号的时候会问一些编程问题，这些编程问题通常是需要有一定编程基础的才可以答对。去挑战一下吧，注册成功后就可以查看一些比较准确的回答了。
