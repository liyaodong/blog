盘点Chrome浏览器的优点集合（选择文艺浏览器，没错）

发表于 2012-08-16   |  

Chrome浏览器作为Google出品的优秀Webkit浏览器引得无数人，包括我在内的喜爱，那么是什么让Chrome浏览器被这么多人接受呢？下面谈谈我的看法。

Chrome浏览器能被这么多人接受，肯定有其优点，那么都有哪些优点呢？

一、反应迅速，渲染效率高

因为是Webkit内核，所以Chrome的速度真的很快！不信么？来看一段视频吧！


现在浏览器的内核引擎，基本上是四分天下：
Trident: IE 以Trident 作为内核引擎;
Gecko: Firefox 是基于 Gecko 开发;
WebKit: Safari, Google Chrome,搜狗浏览器的高速内核,傲游3 基于 Webkit 开发。
Presto: Opera的内核，但由于市场选择问题，主要应用在手机平台–Opera mini
Webkit内核的优势：高效稳定，兼容性好，且源码结构清晰，易于维护。我觉得最主要的是渲染效率高、速度快，画面质量好！WebKit 内核在手机上的应用也十分广泛，例如 Google 的手机 Android、 Apple 的 iPhone, Nokia’s Series 60 browser 等所使用的 Browser 内核引擎，都是基于 WebKit。
Webkit官方网站：http://www.webkit.org/

二、Chrome浏览器具有强大的插件集合

论浏览器插件，最多的应该是Firefox浏览器，这款Mozlia社区出品的开源浏览器同样很出色，有很多插件让浏览器不仅仅是浏览器。同样，Chrome浏览器也有它的插件中心——Chrome Web Store，那这些插件有什么作用？拿以下几个插件来说吧：

①：如意比价淘插件

[caption id=”attachment_621” align=”aligncenter” width=”652” caption=”ChromePlugin”] [/caption]

这款插件可以让你在浏览淘宝、京东、当当等网站的时候给出该商品在其他网站的价格，帮助你选出最便宜的价格，买到最便宜的商品。

②：花瓣网页截图

这款插件可以让你随时采集下你的网页内容，当然，你也可以采用QQ截图，但是如果网页太长了呢？这个时候花瓣整张网页截图就可以很好的帮助你截取长网页图片。并且，采集下来的网页还可以与别人分享，当然也可以看到别人分享的内容。很是不错。

官方网站：http://huaban.com/

还有很多很多插件，大家可以到Chrome应用商店去下载：

https://chrome.google.com/webstore/category/home?hl=zh-cn

三、Chrome强大的快捷键

给出几个我常用的
1.Ctrl + Shift + O 打开书签管理器
2.Ctrl + Shift + B 显示书签栏
3.F12 打开开发人员工具箱
4.Ctrl + F 页面内查找
5.Ctrl + P 打印，实际上是把网页内容保存为PDF格式，就是把网页保存为一个文档，以后方便查看和打印
6.Shift + Esc 打开Chrome浏览器的任务管理器
7.F5 刷新。。。这个应该都会用到
8.Ctrl + H 查看历史记录
我常用的就这么多了，网上有更全版本的快捷键，但是几乎很多都用不到

四、支持HTML5和CSS3等先进标准

虽然HTML5和CSS3这些标准还未发布，但是业界已经广泛使用，Chrome浏览器就是支持这些新标准的先进浏览器。可能很多人都感觉这点没多大用，其实不然，选择Chrome浏览器很大的原因是因为Chrome支持这些新标准新特性。关于HTML5和CSS3可以百度之。

五、拥有强大的开发人员工具箱

F12，你就可以打开开发人员工具箱，也叫审查元素。

这个开发人员工具箱对于非开发者来说有什么用呢？它可以用来查看源代码、修改网页样式（CSS），分析网站性能、监测网页http流量、监测网页占用CPU和内存的大小等等。

假如，你想在土豆或者优酷上下载视频，就会用到这个开发人员工具箱。打开视频后，按下F12，然后再刷新一下，也就是F5，点击network栏目，会看到正在加载的项目，然后点击size标签，也就是根据载入内容大小来排序，网页中视频肯定是最大的，所以加载的视频肯定在最下方，然后找到视频，右键Copy link address就OK了，然后就可以在迅雷中下载了。

这只是开发人员工具箱的一个用处，更重要的是开发人员工具箱是开发者分析网站架构、性能、Bug源与学习、模仿其他网站的一把利器。

六、强大的账户同步功能


申请Google账号并登陆浏览器后就可以同步你的账户信息等等。

这意味着你走到哪里，用其他电脑，只需要登陆你的账号，你的浏览器就又回到你的身边了。包括所有的设置等等，这就是云的优点。

七、防钓鱼防弹窗

Chrome浏览器有庞大的Google支持，所以其有庞大的网页数据库，哪个网站是钓鱼网站，哪个网站有病毒、木马等风险，Chrome都已经知道。当你访问这些网站的时候，Chrome会自动阻止访问，就是这么简单。

八、搜索、地址栏合二为一

在Chrome浏览器里你不需要再设置首页为百度，也不需要打开百度搜索，只需要在地址栏中输入即可搜索。比如搜索宝文理，只需要在地址栏中输入宝文理，OK，就是这么快捷。刚刚安装的用户默认是用Google搜索的，可以在设置中更改搜索引擎。所有的设置都在右上方的扳手里。

说了这么多优点，其实Chrome也是有缺点的。但是相比其他浏览器已经很棒了！可以说比任何一个浏览器都好。

缺点一：无法使用网银，截至目前位置，Chrome浏览器只支持建设银行的网银。所以一旦涉及到网购，还是换回你的IE吧！

缺点二：迅雷支持不足，部分迅雷专用下载链接无法用Chrome直接打开并调用迅雷开始下载，但这只是部分，现在Chrome浏览器对迅雷的支持也越来越好。

我觉得就这么多了，希望对各位刚上手Chrome的朋友有所帮助，大神勿喷，如有错误之处还望指出！
