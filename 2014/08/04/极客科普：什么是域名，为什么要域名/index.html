极客科普：什么是域名，为什么要域名

发表于 2014-08-04   |  


前几天新浪云计算的sinaapp.com域名出故障了，博客和极客协会的网站、查成绩等应用都没法用了。很多同学问我域名是什么，本文就简单介绍一下。

1.什么是域名

域名（Domain Name）就是像bjwlxy.edu.cn这样的网址。那么这一窜字母背后又隐藏这什么呢？我们可以通过windows系统下的命令提示符（DOS）来探索这一过程。
打开你的电脑，按下快捷键Win + R （win就是Ctrl右边那个画着窗子的键）打开运行窗口，输入cmd然后回车，你会看到一个黑框框的存在。在这个黑框框里输入

[code lang=text]
ping bjwlxy.edu.cn
[/code]

然后回车，屏幕下方会出现这一系列看起来很复杂的东西

[code lang=text]
ping bjwlxy.edu.cn
正在 Ping bjwlxy.edu.cn [218.195.117.2] 具有 32 字节的数据:
来自 218.195.117.2 的回复: 字节=32 时间=562ms TTL=128
来自 218.195.117.2 的回复: 字节=32 时间=361ms TTL=128
来自 218.195.117.2 的回复: 字节=32 时间=232ms TTL=128
来自 218.195.117.2 的回复: 字节=32 时间=206ms TTL=128
218.195.117.2 的 Ping 统计信息:
数据包: 已发送 = 4，已接收 = 4，丢失 = 0 (0% 丢失)，
往返行程的估计时间(以毫秒为单位):
最短 = 206ms，最长 = 562ms，平均 = 340ms
[/code]

可以看到我们ping了bjwlxy.edu.cn这个域名之后，系统给我们返回了一个IP地址218.195.117.2如果你直接在浏览器里输入这一串数字，你会发现仍然可以看到我们非常熟(nan)悉(kan)的学校主页。那么域名存在的作用就是将IP地址转化为人类方便记忆的字母。

2.域名后边的.com之类的都有什么意义？

域名可以分为一级域名和二级域名：
bjwlxy.cn这就属于一级域名，而web.bjwlxy.cn则属于二级域名
域名中.cn叫做该域名的后缀，例如baidu.com中的com就是该域名的后缀。需要注意的是域名的后缀不一定都是一个字母组合，咱们学校的域名bjwlxy.edu.cn中的edu.cn也和com一样是一个域名后缀，这并不是一个二级域名。
下面介绍常见的域名后缀意义：

com : Commercial organizations,商业组织,公司

gov : Governmental entities,政府部门

net : Network operations and service centers,网络服务商

org : Other organizations,非盈利组织

edu : Educational institutions,教研机构

info : Infomation,代表一般的信息服务使用，

cn :中国国家顶级域名

com.cn :中国公司和商业组织域名

net.cn :中国网络服务机构域名

gov.cn :中国政府机构域名

org.cn :中国非盈利组织域名

edu.cn :中国的教研机构域名

3.我如何注册一个域名

域名的注册是到域名注册商来注册的，国内的域名注册商有

  1. 万网
  2. 美橙互联
  3. 新网
  4. 花生壳
  5. 阿里云

国外的域名注册商有

  1. Godaddy（狗爹）
  2. name
  3. enom
  4. Ipower

域名注册商之间的区别是价格、服务、稳定性、域名转入、转出政策等，我都是在Godaddy注册域名的，因为Godaddy是这个星球上最大的域名注册商，价格也相对还可以，唯一缺点就是在他们的英文网站里很难找到支付宝的支付链接。不过网上有一些优惠码和教程可以自己百度一下。

4.域名的价格

域名价格是变动的，但幅度不是很大。以Godaddy的域名价格为例

com ¥71.99

net ¥62.99

info ¥18.99

需要注意的是，域名首次注册的费用和续费的费用往往不同，一般情况下续费会贵点。另外域名购买的时候一般是可以找到优惠码的，上图这个是原价。优惠码有时候可以节约20%，这些都需要你在买域名之前提前了解好。
