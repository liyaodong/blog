想要安装Win8的同学看过来

发表于 2013-01-19   |  

Win8发布也有一段时间了，兼容性等也不再像以前那么差了。可以说，差不多已经成熟了。如果你还是Win7或者xp想要升级Win8，但是怕兼容性不好。先不要果断放弃升级，因为在我测试的几台电脑中，Win8的兼容性还是很不错的。

就目前测试的几台电脑来说，Win8的兼容性还是很不多的。各种驱动都可以完美的打上，常用的软件、游戏也可以流畅运行。而且，最重要的是，开机速度着实有很大的提升。如果你心动了，那么请继续往下看，一步一步教你升级Win8。

1、升级Win8前的准备

因为大家都怕麻烦，所以在此给大家介绍一种最简单的升级办法。简单到你不能想象。首先你要明确你想安装32位还是64的Win8，如果你的电脑内存条4G，处理器是i5及其以上，那么建议安装64位。如果是2G内存的话，建议装32位系统。因为2G内存跑64位会有一点吃力。

确定好你要安装的位数之后，下来就是要下载系统了。下载系统推荐MSDN的官方纯净镜像，正版，稳定。

宝文理校园网用户下载方法：

打开某个文件夹，将下面的地址复制到文件夹的地址栏内然后回车

\218.195.112.3\rm\2012年新片\win 7 8

稍后片刻，会让你输入账号密码。账号密码都是wlzx

然后找到你所需要的版本，推荐64位专业版Win8

注：这种方法只适用于校园内网用户

外网用户下载方法：

http://www.xp85.com/html/Win8-Por-VL.html

同理，找到你所需要的版本，推荐64位专业版Win8

下载好系统之后，使用Winrar解压到你的电脑某盘的根目录里，但是不能解压到C盘。加入你把文件下载到E盘下了，你只需要在文件上点击右键-解压到当前文件夹。

然后你会看到有很多的文件，其中一个叫做Setup.exe这个就是Win8的系统安装文件。双击Setup.exe就可以开始安装了。在安装前还需要准备一个东西，就是Win8的安装码。下面给出

Win8安装码：

Windows 8 MSDN 简体中文版 x86（包含专业版和Core版）
安装密匙    NG4HW-VH26C-733KW-K6F98-J8CK4
Windows 8 MSDN  简体中文版 x64（包含专业版和Core版）
安装密匙    NG4HW-VH26C-733KW-K6F98-J8CK4
Windows 8  Pro VL 简体中文专业版 x86（批量授权版）
安装密匙    NG4HW-VH26C-733KW-K6F98-J8CK4
Windows 8  Pro VL 简体中文专业版 x64（批量授权版）
安装密匙    NG4HW-VH26C-733KW-K6F98-J8CK4
Windows 8 Enterprise 简体中文企业版 x86
安装密匙    32JNW-9KQ84-P47T8-D8GGY-CWCK7
Windows 8 Enterprise 简体中文企业版 x64
安装密匙    32JNW-9KQ84-P47T8-D8GGY-CWCK7

这个最好记一下，如果你是专业版32位就是倒数第四个，x86就是32位的意思。很奇怪为什么不是x32吧？这个自己百度吧，很有意思的一个计算机历史问题。
记号了安装码之后就可以在Setup上点击右键-使用管理员身份开始安装了。如果运行Setup没有任何错误，那么恭喜你可以以最简单，最安全的方法升级Win8。如果提示你系统不兼容，那么也别急。原因可能是你目前的系统是32位的，但是你要安装64位的Win8。64位的文件无法在32位系统上运行。

这个时候你就需要下载一个系统引导工具了。

下载地址：http://pan.baidu.com/share/link?shareid=172825&uk=3171893396

这个工具下载下来后，以管理员身份运行之。然后会引导你进行安装，安装好之后，重启电脑。选择第二个开机启动项同样可以开始安装Win8。而且这种方式可以格式化你的C盘，也就是可以安装一个纯净的Win8而不去管Win7的文件。不过推荐初级用户选择升级，安全一点。

使用系统引导工具安装之前，切记要将系统解压到某个盘的根目录，不然这个引导工具是不能安装成功的。

具体的安装过程：

http://www.ithome.com/html/win8/23040.htm

这里有详细的安装过程，这就不罗嗦了。安装完成后，校园网的用户需要注意了。

Win8锐捷安装方法：

Win8下直接安装锐捷是不能安装上的，在Win8下安装锐捷需要在安装包，注意是安装包上点击右键-属性，选择兼容性选项卡，勾选以兼容模式运行此程序。兼容模式选择Windows vista （Service Pack 1）然后确定。再以管理员身份安装锐捷就可以安装上了。至于为什么是vista sp1的兼容模式，因为这个是一个一个试出来的。在Win8下如果有的软件无法运行，可以尝试用兼容模式运行，你会有意想不到的收获。

Win8的破解：

在天朝，不要怕Win8安装上之后无法破解，我大天朝人才济济，小小的微软系统破解有什么难度，神马地沟油三聚氰胺苏丹红假鸡蛋都可以造得出来，何况破解一个小小的系统？

Win8激活破解软件下载：http://pan.baidu.com/share/link?shareid=172839&uk=3171893396

同样以管理员身份运行之就可以了。

系统安装好之后，使用NT6引导的同学，需要再次以管理员方式运行这个系统引导器，不过不一样的是这次选择的是卸载。这样你的Win8就不会有那么多的启动项了。

之前安装Win7都是通过使用PE然后用WIM工具的，但是Win8的专业版没有了install.wim所以也不能通过PE下的wim安装，综合来看使用NT6 HDD安装还是一个比较简单的方法。不需要U盘，不需要光盘，省事很多。Win8安装到此结束，如有疑问欢迎评论或者自行百度。

推荐一个入门、精通Win8的方法：http://bbs.pcbeta.com/windows8/ 远景论坛的WIN8板块，流B人物很多，教程很全面。远景威武霸气！
