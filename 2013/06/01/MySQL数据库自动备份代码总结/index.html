MySQL数据库自动备份代码总结

发表于 2013-06-01   |  

折腾了近一个小时，终于搞定了。再也不怕因为自己的误操作搞坏数据库了。虽然没啥技术含量，但是实用就行。

首先明确下前提条件：Win Server 2003 、 Phpnow提供的绿色套件，注意是绿色的，所以停止和开启MySQL的方法会有所不同。

直接上代码吧：
@echo off call My_Stop.cmd start d:\progra~1\WinRAR\winrar.exe a /agYY-MM-DD /k /r /s /ibck d:\dbbackup\dbbackup.rar d:\PHPnow-1.5.6\MySQL-5.1.50\data\dhy d:\PHPnow-1.5.6\MySQL-5.1.50\data\jgx d:\PHPnow-1.5.6\MySQL-5.1.50\data\jxjy d:\PHPnow-1.5.6\MySQL-5.1.50\data\tuanwei d:\PHPnow-1.5.6\MySQL-5.1.50\data\yb d:\PHPnow-1.5.6\MySQL-5.1.50\data\zfx call My_Start.cmd echo ======================================== &gt;&gt; d:\dbbackup\log.txt echo date： %date% &gt;&gt; d:\dbbackup\log.txt echo time：%time:~0,5% &gt;&gt; d:\dbbackup\log.txt echo ======================================== &gt;&gt; d:\dbbackup\log.txt exit

首先在D盘建立dbbackup文件夹，然后把代码写一个bat放到Phpnow\pncmds这个文件夹下，为什么是这个文件夹？因为我的My_stop.cmd在这里。然后在计划任务里添加一个计划任务，每天凌晨运行，你的数据库每天就会自动备份到dbbackup文件夹下。并且是以dbbackup13-06-01这样以日期为命名的压缩文件。这里还要感谢Winrar这个强大工具，有兴趣了可以搜索一下Winrar命令行。一般情况下，这样备份就足够了，如果你还不放心，你可以写个代码自动把备份文件发送到邮箱或者FTP或者SVN同步之类的，随便你了。当然，这不在本文讨论范围之内。整体来说，没啥技术含量，很古老的文件备份，没有使用到MySQL的高级功能，以后慢慢完善，先凑活用吧。
